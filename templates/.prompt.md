## output formatting

docstrings / comments should be generated concise, using lowercase, shortforms, & improper grammar to shorten the length of comment.

for example, this is not preferred:
```
// Helper function to get the playwright toolsets for dynamic usage and validation (e.g. do this)
// saves all records from the linked list to the db file
```

this is preferred:
```
// helper func to get playwright toolsets for dynamic usage & validation (eg do this)
// save all records from linked list to db
```

diagrams should be in mermaid syntax. Math formulas (if any) should be in LaTeX syntax.
